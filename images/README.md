# Gingerbread Edinburgh & Lothian — Images Folder

Upload these image/video files to this folder on GitHub.
All files will display automatically once uploaded. No code changes needed.

---

## REQUIRED

- hero-video.mp4
  → 8-second looping promo video for the home page hero section
  → If missing, a dark green gradient is shown instead (site still looks fine)

- logo.png
  → Gingerbread logo (transparent background preferred)
  → Currently using Gingerbread LOGO.jpg from the root folder — replace if you have a better version

---

## HOME PAGE

- about.jpg
  → A photo of children at one of your clubs, or staff with children
  → Shown in the "Supporting Single Parent Families Since 1985" section on the home page
  → Replaces the placeholder emoji — recommended size: 800x600px

## NOTES

- Recommended image formats: JPG or PNG
- Recommended max file size: under 2MB per image (for fast loading)
- Hero video: MP4 format, ideally under 15MB, 8–15 seconds, looping
- All images should avoid showing children's faces clearly without parental consent
- You do not need to upload all images at once — the site handles missing images gracefully
